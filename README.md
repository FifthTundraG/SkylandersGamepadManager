# Skylanders Gamepad Manager
The first and only cross-platform application for using the Skylanders Trap Team mobile gamepad on Windows, macOS, or Linux.

## Platform Support
**NOTE: THIS PLATFORM SUPPORT IS INACCURATE AS OF WRITING**

- ✅ Linux: Fully supported and working
- 🚧 Windows: In progress
- ⛔ macOS: Planned but not implemented

## Usage
Open the application, then connect a new gamepad over your system's Bluetooth settings. After connecting, it should appear in SkylandersGamepadManager and a new virtual input device will be created that can be used in any software.
